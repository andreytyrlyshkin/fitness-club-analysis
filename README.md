# Анализ клиентской базы фитнес-клуба

## О проекте
Аналитический проект, посвящённый исследованию клиентской базы сети фитнес-клубов.  
Включает три направления анализа: обработку возрастных данных, EDA и сегментацию клиентов.

## [Данные](https://github.com/andreytyrlyshkin/fitness-club-analysis/tree/master/data)
В работе использованы синтетические датасеты, имитирующие реальные данные фитнес-клуба:
- `users_info.csv` — информация о пользователях (id, город, возраст, пол)
- `payments_monthly.csv` — помесячные данные о тренировках и платежах
- `merged_data.csv` — объединённый датасет для анализа

## Инструменты
- Python 3
- pandas — обработка и анализ данных
- numpy — математические вычисления
- matplotlib / seaborn — визуализация
- Jupyter Notebook — среда

## [Вычисления](https://github.com/andreytyrlyshkin/fitness-club-analysis/tree/master/notebooks)
- `01_age_analysis.ipynb` — анализ возраста с обработкой выбросов
- `02_eda_analysis.ipynb` — исследовательский анализ данных
- `03_segmentation_analysis.ipynb` — сегментация клиентской базы

## Как открыть
1. Скачайте репозиторий
2. Откройте файлы `01_age_analysis.ipynb`, `02_eda_analysis.ipynb`, `03_segmentation_analysis.ipynb` в Jupyter Notebook или JupyterLab.
3. Выполняйте ячейки последовательно.

## Этапы анализа

### 1. Анализ возрастного распределения
- Расчёт среднего возраста с фильтрацией (до 50 лет) и без
- Разработка функции `calc_avg_age()` с обработкой значений с порогом
- Применение метода IQR для выявления и исключения выбросов

**Ключевой вывод:** средний возраст клиентской базы (без учёта выбросов) составляет около 30 лет.

### 2. Исследовательский анализ данных (EDA)
- Проверка структуры данных: размер, типы, пропуски
- Удаление пропусков
- Визуализация распределения возраста
- Анализ распределения пользователей по городам и полу
- Расчёт доли женщин в разрезе городов

**Ключевой вывод:** среди всех городов наибольшая доля женщин-клиентов зафиксирована в Москве (>60%).

### 3. Сегментационный анализ
- Выделен сегмент пользователей с преобладанием индивидуальных тренировок (>70%)
- Для сегмента рассчитаны: средний чек, среднее число тренировок, распределение по городам и полу
- Определены топ-10 самых активных клиентов
- Построена динамика клиентской базы по месяцам (новые, ушедшие, активные клиенты)

**Ключевые выводы:**
- В сегменте индивидуальных тренировок преобладают женщины (~60%)
- Пик оттока клиентов приходится на октябрь-декабрь
- Большинство самых активных клиентов сосредоточено в Москве
